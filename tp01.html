5. investigar que hacen las etiquetas span, i, b, a

span

es un contenedor en linea. sirve para aplicar estilo al texto o agrupar elementos en linea

etiqueta "i"

muestra el texto marcado con un estilo en cursiva o italica

etiqueta "b"

Indica que el texto debe ser representado con una variable bold, o negrita,de la tipografía que se esté usando.

etiqueta "a"

crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

6.en la estrcutura basica del documento html,¿como definiria html, head y body?

HTML

El elemento HTML (o elemento HTML raiz) representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.

HEAD

El elemento HEAD provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

BODY

El elemento de HTML representa el contenido de un documento HTML. Solo puede haber un elemento en un documento
